# llm_process

## 能力描述
通用的 LLM 处理脚本，接受用户输入，返回 LLM 输出结果。

## 脚本说明

### llm_process.py
**功能**：调用 LLM 处理输入内容

**入参**：
| 字段 | 类型 | 必填 | 描述 |
|------|------|------|------|
| input | string | 是 | 要发送给 LLM 的输入内容 |

**出参**：
| 字段 | 类型 | 描述 |
|------|------|------|
| output | string | LLM 的输出结果 |

**调用示例**：
```bash
python skills/llm_process/scripts/llm_process.py '{"input": "请帮我总结以下内容：今天天气很好，阳光明媚。"}'
```

**输出示例**：
```json
{"output": "今天天气晴朗，阳光充足。"}
```

## 使用场景
- 对比两次输出结果
- 总结或分析数据
- 格式化输出
- 任何需要 LLM 处理的场景
